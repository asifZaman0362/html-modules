HTML MODULES

A collection of simple and easy to use modules for modern websites.

static gallery

A gallery view with images loaded from the server. Clicking on an image opens it in another tab.

image slides

A slideshow view of images loaded from the server with different transition effects.

Navbar

A mobile friendly sticky navigation bar. Everyone needs one of those huh!
